Características

React tiene una distinta serie de características, entre las que se incluyen:

  • Lenguaje declarativo. Los desarrolladores diseñan vistas para cada estado de una aplicación, y React actualiza y renderiza los componentes cuando cambian los datos.
  • Componentes
    • Reutilizables
    • Deben formarse en la carpeta SRC
  • DOM virtual.
    1. React crea un caché de estructura de datos en la memoria
    2. Calcula diferencias resultantes
    3. Actualiza el DOM
  • Ciclos de vida. Son métodos que se ejecutan cuando pasan cosas comunes con el componente, que nos permiten suscribir acciones cuando se produce una inicialización, se recibe la devolución de una promesa, etc.
  • Hooks. Funciones que permiten a los desarrolladores "engancharse" a las características de ciclo de vida y estado de React de los componentes de función
